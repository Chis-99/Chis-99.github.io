Data Protection Impact Assessment (DPIA)

Tags: Privacy • Risk • Compliance • Governance

Summary

I conducted a DPIA for a school environment to identify high-risk personal data processing, assess impacts to students/staff, and document mitigations. All examples are simulated/redacted.

Jamaica DPA GDPR Concepts Risk Scoring Data Flow Mapping

Problem

Schools process sensitive personal data (students, guardians, staff) across multiple systems. The goal was to identify where privacy risks exist and define realistic mitigations and controls.

Approach

  • Defined processing activities (admissions, attendance, grades, health, photos/media)
  • Mapped data flows (collection → storage → sharing → retention/disposal)
  • Scored risks (Likelihood × Impact) and prioritized high-risk items
  • Mapped mitigations to administrative + technical controls

Key Deliverables

  • DPIA Report (PDF)
  • Data Flow Diagram (PNG/PDF)
  • Risk Register (CSV/Excel)
  • Compliance Checklist (PDF)
